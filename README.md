## 📋 Overview

This repository contains a comprehensive, implementation-ready Standard Operating Procedure (SOP) for managing the lifecycle of leased computer equipment under the Asset Finance Solution contract. It provides end-to-end guidance from contract award and kick-off through business-as-usual operations to contract close-out.

**Service Provider:** COCRE8 Technology Solutions (or equivalent)
**Contract Duration:** 5 years
**Scope:** Leasing of ICT assets (computers, laptops, peripherals)
